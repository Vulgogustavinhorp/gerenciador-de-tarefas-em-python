# Gerenciador de Tarefas em Python

Este projeto é um simples gerenciador de tarefas desenvolvido em Python. Ele permite adicionar, listar, remover e marcar tarefas como concluídas, ajudando a organizar suas atividades diárias.

## Funcionalidades

- Adicionar novas tarefas
- Listar tarefas pendentes e concluídas
- Marcar tarefas como concluídas
- Remover tarefas
